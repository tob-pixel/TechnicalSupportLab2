Information system for mobile fitness support for a student on the iOS platform

Mobile fitness tracking app for iOS

Short description

The project includes an algorithm for dynamic adaptation of vantagement. The system not only implements the plan, but changes the number of repetitions at the onset of training based on a subjective assessment of the importance of the previous exercise (RPE - Rate of Perceived Exertion).

All calculations and generation of recommendations are carried out on the device using Apple Core ML frameworks, which solves the problem of longevity on the Internet and subscriptions.

Meta surveillance

Vector development of the project aimed at the thorough implementation of the system of dynamic adaptation to attract, update information on new models on the Apple platform..
