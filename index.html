Mobile fitness support information system for the user on the iOS platform

Mobile fitness tracking app for iOS

Short description

The project is a dynamic load adaptation algorithm. The system does not just offer a plan, but changes the weight or number of repetitions for the next workout based on the subjective assessment of the difficulty of the previous session (RPE - Rate of Perceived Exertion).

All calculations and recommendation generation occur on the device using Apple Core ML frameworks, which solves the problem of dependence on the Internet and subscriptions.

Research objective

The vector of project development is aimed at improving and implementing a system of dynamic load adaptation and generalization of information using language models on the Apple platform.

Main tasks

  • Analysis of the subject area and existing solutions.
  • Designing the architecture of an information system.
  • Software development and testing.
  • Evaluation of the effectiveness of the results obtained.

Research object

Expected results

It is planned to obtain a convenient system for independent detailed tracking of physical exercise progress.

The user should enjoy using it.
